# Skill: fundamental-analysis

金融产品基本面分析。使用搜索工具获取股票、ETF、基金、加密币的财务数据、财报信息、机构评级、估值数据等，生成 Markdown 格式的基本面分析报告。

## When to Use

- 用户要求分析股票、ETF、基金、加密币的基本面
- 需要获取财务数据、财报信息、机构评级
- 需要进行估值分析（市盈率、市净率、PEG等）
- 要求生成基本面分析报告

## 基本面分析框架

### 1. 财务分析

| 指标 | 说明 | 评估标准 |
|------|------|----------|
| 营收 | 营业收入规模 | 增长>20%为优秀 |
| 净利润 | 盈利水平 | 增长>20%为优秀 |
| 毛利率 | 产品盈利能力 | >30%为高毛利 |
| 净利率 | 经营效率 | >15%为优秀 |
| ROE | 净资产收益率 | >15%为优秀 |
| 资产负债率 | 财务杠杆 | <50%为健康 |

### 2. 估值分析

| 指标 | 说明 | 评估标准 |
|------|------|----------|
| PE (市盈率) | 股价/每股收益 | 低于行业平均为低估 |
| PB (市净率) | 股价/每股净资产 | 低于3为合理 |
| PEG | PE/净利润增长率 | <1为低估 |
| 股息率 | 分红/股价 | >3%为高股息 |

### 3. 机构观点

| 类型 | 说明 |
|------|------|
| 评级 | 买入、增持、中性、减持、卖出 |
| 目标价 | 机构预测的未来价格 |
| 研报 | 分析师的详细分析报告 |

### 4. 行业分析

| 维度 | 说明 |
|------|------|
| 行业地位 | 龙头、细分龙头 |
| 行业周期 | 成长期、成熟期、衰退期 |
| 政策影响 | 受益还是受损 |

## 注意事项

1. **数据来源**: 搜索结果来自公开网络，需核实准确性
2. **时效性**: 财务数据有滞后性，**优先使用离当前最近的一个季度数据**，包括最新的季报、中报或年报
3. **季度数据优先**: 在搜索财报时，优先搜索"最新季度"、"Q1/Q2/Q3/Q4"、"中报"、"年报"等关键词，确保使用最新数据
4. **投资风险**: 基本面分析仅供参考，不构成投资建议
5. **合规要求**: 遵守证券法规，不传播未公开信息
