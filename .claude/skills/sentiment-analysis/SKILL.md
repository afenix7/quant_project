# Skill: sentiment-analysis

A股市场情绪分析。参考"炒股养家"情绪分析框架，通过分析成交量、价格波动、舆情热度、技术面指标等维度，生成股票情绪分析报告。

## When to Use

- 用户要求进行股票情绪分析
- 需要分析市场情绪周期（启动期、发酵期、高潮期、退潮期）
- 需要分析股吧、财经社区的舆情情绪
- 需要判断市场多空对比和资金流向

## Prerequisites

本 Skill 主要使用以下工具获取数据：
- **agent-browser**: 浏览器自动化工具，用于访问东方财富股吧
- **web_search**: MiniMax MCP 搜索工具，用于补充搜索机构研报和新闻
- 东方财富网：quote.eastmoney.com, guba.eastmoney.com, data.eastmoney.com

**重要提示**：
- 评论数据必须是当天或最近一周的
- 优先使用 agent-browser 获取最新实时评论

## 情绪分析框架

### 一、直接观察市场情绪的指标

#### 1. 成交量与换手率

| 指标 | 情绪信号 |
|------|----------|
| 突然放量（历史均量2-3倍） | 可能反映资金抢筹（利好）或恐慌抛售（利空） |
| 缩量上涨 | 筹码锁定度高，但需警惕流动性不足后的反转风险 |
| 换手率超50%（次新股） | 情绪过热信号 |
| 低位换手率骤增 | 题材启动信号 |

#### 2. 价格波动特征

| 指标 | 情绪信号 |
|------|----------|
| 日内振幅超10% | 市场情绪剧烈分歧 |
| 天地板/地天板 | 极端情绪表现 |
| 涨停封单量级（如百万手） | 资金信心体现 |
| 盘中频繁撤单 | 筹码松动预警 |
| 龙虎榜游资席位买入 | 带动跟风情绪 |

#### 3. 新闻与舆情热度

| 指标 | 情绪信号 |
|------|----------|
| 突发政策利好 | 情绪脉冲 |
| 社交媒体发酵（股吧） | 舆情热度上升 |
| 财经自媒体观点一致性 | 过度乐观/悲观预警 |
| 讨论量激增 | 关注度达到高点 |

### 二、技术面情绪验证

#### 1. 极端技术形态

| 指标 | 情绪信号 |
|------|----------|
| RSI >80 | 超买，情绪可能过热 |
| RSI <20 | 超卖，可能存在错杀机会 |
| KDJ钝化 | 情绪极端 |
| MACD背离 | 情绪可能透支 |
| 跳空高开不回补 | 短期支撑位 |

#### 2. 资金流向验证

| 指标 | 情绪信号 |
|------|----------|
| 北向资金单日净流入超5亿 | 可能引发跟风效应 |
| 大单买入占比超60% | 主力控盘 |
| 主力资金持续净流入 | 多头情绪 |

### 三、市场情绪周期判断

| 阶段 | 特征 |
|------|------|
| 启动期 | 龙头股悄然连板，市场关注度低 |
| 发酵期 | 板块跟风股补涨，媒体开始报道 |
| 高潮期 | 散户大量入场，成交量暴增，缩量加速走势 |
| 退潮期 | 龙头滞涨，后排股大跌，跌停家数增加 |

### 四、情绪极端点信号

| 信号类型 | 特征 |
|----------|------|
| 恐慌性抛售 | 百股跌停、成交量骤缩、分析师集体看空 |
| 贪婪顶点 | 新股民开户数激增、分析师喊出"万点论"、散户一致性看涨 |

### 五、情绪陷阱规避

1. **避免过度解读单一信号**：如仅凭利好消息追涨，忽略技术面超买
2. **警惕一致性预期**：当所有散户都认为"必涨"时，往往是主力出货时机
3. **情绪与基本面背离**：题材炒作股若缺乏业绩支撑，情绪退潮后可能腰斩

## 数据获取方法

### 1. 使用 agent-browser 获取股吧最新舆情

```bash
# 1. 打开股吧页面
agent-browser open https://guba.eastmoney.com/list,{股票代码}.html

# 2. 获取页面快照，提取最新评论
agent-browser snapshot -c

# 从快照中提取：
# - 评论标题和内容
# - 发布时间
# - 作者信息
```

### 2. 股票基本信息和资金流向

```bash
# 使用 web_search 搜索最新行情和资金数据
web_search(query="股票代码 股票名称 股价 资金流向 换手率 2026")
```

### 3. 舆情数据筛选标准

**必须满足以下条件：**
- 评论内容：与该股票相关，只取标题
- 样本数量：股吧里首页全部评论，不翻页

**情绪分类：**
- positive：看多、看涨、乐观
- negative：看空、看跌、风险提示
- neutral：中性、疑问、事实陈述

## 报告生成

### 输出目录

所有报告保存到: `~/uni-web/frontend-html/finance-analysis/reports/`

### 舆情数据保存

获取到的股吧和雪球舆情需要保存到 `data/` 目录，JSON 格式如下：

**文件命名规则**: `{股票代码}-comments-{YYYYMMDD}.json`

例如: `300666-comments-20260218.json`

**JSON 结构**:
```json
{
  "stock_code": "300666",
  "date": "2026-02-18",
  "sources": {
    "guba": {
      "url": "https://guba.eastmoney.com/list,300666.html",
      "comments": [
        {
          "content": "评论内容",
          "sentiment": "positive/neutral/negative",
          "timestamp": "2026-02-18 10:30:00"
        }
      ],
      "summary": {
        "positive_count": 10,
        "neutral_count": 5,
        "negative_count": 3
      }
    }
  }
}
```

### 文件命名规则

```
{股票名称}情绪分析报告_{YYYYMMDD}.md
```

例如: `江丰电子情绪分析报告_20260218.md`

## 使用示例

### 分析A股江丰电子 (300666)

1. 使用 agent-browser 获取股吧最新舆情
2. 使用 web_search 补充搜索机构研报
3. 搜索最新行情和资金数据
4. 保存舆情数据到 JSON（{股票代码}-comments-{YYYYMMDD}.json）
5. 分析情绪指标
6. 生成情绪分析报告

```bash
# 1. 使用 agent-browser 获取股吧最新舆情
agent-browser open https://guba.eastmoney.com/list,300666.html
agent-browser snapshot -c
# 从快照中提取今天（02-18）的最新评论

# 2. 使用 web_search 补充搜索机构研报
web_search(query="300666 江丰电子 研报 评级 买入 目标价")

# 3. 搜索最新行情和资金数据
web_search(query="300666 江丰电子 股价 资金流向 换手率 2026")
```

### 完整工作流程

```
1. 打开股吧：agent-browser open https://guba.eastmoney.com/list,{股票代码}.html
2. 获取快照：agent-browser snapshot -c
3. 提取评论：筛选今天或最近7天的评论
4. 保存数据：写入 data/{股票代码}-comments-{YYYYMMDD}.json
5. 搜索补充：web_search 搜索机构研报
6. 生成报告：按模板生成情绪分析报告
```

## 注意事项

1. **数据时效性要求（重要）**：
   - 资金流向和行情数据需结合最新交易日
   - 在报告中注明数据获取时间

2. **情绪欺骗性**：主力资金可能利用情绪误导散户，需多维度验证
3. **风险警示**：情绪分析仅供参考，不能作为唯一投资依据
4. **合规要求**：遵守证券法规，不传播未公开信息
5. **工具使用优先级**：
   - 优先使用 agent-browser 获取实时评论
   - 机构研报用 web_search 补充搜索
